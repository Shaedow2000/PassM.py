## I-/ Installation:
### 1) Clone the repo:
Clone the Github repository:
```bash
git clone 'https://github.com/Shaedow2000/PassM.py.git'
```
Or the Codeberg repository:
```bash
git clone 'https://codeberg.org/Shaedow2000/PassM.py.git'
```
> Both repositories are the same.
### 2) Change directories:
Change the current direcotry to the root of the downloaded PassM folder:
```bash
cd PassM.py
```
### 3) Install the requirements:
Install all the required packages:
```bash
pip install -r requirements.txt
```
### 4) Create executable command:
Now to execute the app, create an executable command to start the app:
```bash
pip install -e .
```
### 5) Execute the program:
Now, just execute the command and start the app:
```bash
passm
```
---
> [!IMPORTANT]
> If you use an operating system like *Arch Linux*, use the `--break-system-packages` tag in the end of the pip install commands.
